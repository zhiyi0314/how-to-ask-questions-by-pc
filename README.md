# 电脑维修提问生存指南

## 在你开口之前，先他妈的想清楚

### 0. 灵魂拷问（必答）
在浪费别人时间之前，先回答这几个问题：

- [ ] 你**重启**过吗？（不是关机再开，是**重启**）
- [ ] 你**百度/Google**过了吗？
- [ ] 问题能**复现**吗？还是偶发的玄学事件？
- [ ] 你最近**动了什么**？装软件？插U盘？下载小电影？

如果上面有任何一条没做，**滚回去做了再来**。

---

## 1. 描述问题的正确姿势

### ❌ 错误示范（找死型）
> "我电脑坏了"
> 
> "网连不上"
> 
> "打不开"
> 
> "刚才还好好的"

**你他妈说了跟没说一样。**

### ✅ 正确示范（人话型）
> "Win11 23H2，开机后WiFi图标显示红叉，重启无效，设备管理器里网卡有黄色感叹号，错误代码43"

**信息密度 = 尊重程度**

## 2. 必须提供的信息清单

| 项目 | 例子 | 不说人话的后果 |
|------|------|---------------|
| **系统版本** | Win10 22H2 / macOS 14.3 | 我按Win11的方法教你，你发现用不了，浪费双方20分钟 |
| **硬件型号** | 联想拯救者Y7000P 2023 | "笔记本"有上万种，你说的是哪种？ |
| **问题现象** | 蓝屏，代码CRITICAL_PROCESS_DIED | "卡住了"是啥意思？死机？慢？还是你智商卡住了？ |
| **触发条件** | 每次打开Photoshop必崩 | 偶发问题需要不同的排查思路 |
| **已尝试的解决** | 重装驱动、杀毒、 sacrifices to the old gods | 避免我让你重复你已经做过的无用功 |

## 3. 拍照与截图规范

### 手机拍屏幕的铁律：
1. **对焦** - 糊成毕加索油画的我看不懂
2. **光线** - 别背光，别反光，别拍成恐怖片
3. **全屏** - 我要看**整个屏幕**，不是只拍错误弹窗，上下文很重要
4. **别拍键盘** - 除非问题是键盘真的着火了

### 截图工具推荐（免费的，不用谢）：
- Win: Win+Shift+S 或 ShareX
- Mac: Cmd+Shift+4
- 手机：电源键+音量键，别他妈用另一个手机拍

---

## 4. 禁止行为（违者直接拉黑）

### 🚫 高频傻逼问题 TOP 5

1. **"在吗？"**
   - 有话直说，我不在，我永远不在，但我会看消息。直接发问题。

2. **"你能帮我看看吗？很快的"**
   - 没有"很快"的电脑问题。你感觉很快是因为你不懂。

3. **"我按你说的做了，还是不行"**
   - **详细描述**怎么做的，出现了什么**新**的错误。这句话等于"医生我吃药了还是难受"，医生怎么知道你怎么吃的？

4. **"你是不是觉得我很笨？"**
   - 不，我觉得你很忙，忙到连百度都懒得用。

5. **"淘宝修电脑的说是XX问题"**
   - 淘宝客服的目标是让你下单，不是解决问题。别拿他们的诊断来质疑我。

## 5. 远程协助 etiquette

如果要让我远程控制你的电脑：

- **提前关掉色情内容** - 我真的不想看到
- **准备好管理员密码** - 别他妈临时打电话问你妈
- **坐在电脑前** - 别去吃饭让我对着桌面发呆
- **别同时找两个人** - 多个"医生"一起动刀，死得更快

---

## 6. 费用与尊重

**免费帮忙 ≠ 24小时客服**

- 晚上11点后发消息，最好准备好红包
- 解决了问题，说句"谢谢"是最低要求
- "请喝奶茶"是基本礼仪，不是可选项目

---

## 总结：提问的艺术公式
好的提问 = 具体环境 + 明确现象 + 已做尝试 + 截图/照片

**记住：你节省我的时间 = 你更快解决问题**

**最后警告：如果你的问题让我感觉在教猴子用电脑，我会直接推荐你去电脑城送人头。那里的人很"热情"，而且一定比你聪明（在骗钱方面）。**
