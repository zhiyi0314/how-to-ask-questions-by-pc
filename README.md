
### ✅ 好例子：
> "电脑开机后一直黑屏，风扇在转，屏幕完全没反应，已经强制关机重启两次还是黑屏。"

**（我看到立马知道是显示问题，直接给方案）**

### ❌ 烂例子：
> "我电脑坏了"
> 
> "怎么办"
> 
> [十分钟后]
> 
> "就是开不了"
> 
> [二十分钟后]
> 
> "屏幕是黑的"

**（这种我要问：是按了电源没反应？还是进了系统黑屏？还是蓝屏？你等不等的起？）**

---

## 3. 描述>猜测

**别说你的诊断，说你的现象。**

### ❌ 错误：
> "我显卡好像烧了"  
> "是不是中病毒了"  
> "可能内存松了"

**你知道个屁。直接说：**
- 屏幕有什么（花屏/黑屏/蓝屏/报错字）
- 声音有什么（风扇狂转/滴滴叫/没声音）
- 你干了什么（摔了/进水/更新系统）

---

## 4. 等待成本说明

**为什么逼你一次说完：**

| 你的做法 | 耗时 |
|---------|------|
| 你："电脑坏了" | 0秒 |
| 我："具体？" | 等5分钟 |
| 你："开不了机" | 等5分钟 |
| 我："电源灯亮吗？" | 等5分钟 |
| 你："亮的" | 等5分钟 |
| 我："风扇转吗？" | ... |

**总计：30分钟过去了，我还没开始修。**

### 一次性说完：
| 你的做法 | 耗时 |
|---------|------|
| 你："按电源键灯亮风扇转，屏幕不亮，没摔过" | 30秒 |
| 我："拔电池重试/外接显示器测试..." | **直接解决** |

---

## 5. 极简检查（做不做随你）

如果愿意，先试试这个，然后**把结果告诉我**：
1. 重启（不是关机再开，是选"重启"）
2. 拍张屏幕照片（对焦清楚，别手抖）

---

## 总结：发消息前自检

- [ ] 删掉"在吗/忙吗/在不在"
- [ ] 说了**现象**（看到什么）而不是**结论**（你觉得是什么）
- [ ] 说了**之前发生了什么**（装软件/更新/摔了）
- [ ] 说了**试过什么**（重启/没动）

**做不到以上，就等着我在忙完正事后（可能几小时后）再一句一句问你。**

# 电脑维修提问生存指南

## 在你开口之前，先他妈的想清楚

### 0. 灵魂拷问（必答）
在浪费别人时间之前，先回答这几个问题：

- [ ] 你**重启**过吗？（不是关机再开，是**重启**）
- [ ] 你**百度/Google**过了吗？
- [ ] 问题能**复现**吗？还是偶发的玄学事件？
- [ ] 你最近**动了什么**？装软件？插U盘？下载小电影？

如果上面有任何一条没做，**滚回去做了再来**。

## 1. 描述问题的正确姿势

### ❌ 错误示范（找死型）
> "我电脑坏了"
> 
> "网连不上"
> 
> "打不开"
> 
> "刚才还好好的"
> 
> "在吗＂

**你他妈说了跟没说一样。**

### ✅ 正确示范（人话型）
> "电脑/手机 在做过什么事后发生了什么，最后导致了什么（有截图和视频，提供图片和视频"

**信息密度 = 尊重程度**

## 2. 必须提供的信息清单

| 项目 | 例子 | 不说人话的后果 |
|------|------|---------------|
| **系统版本** | Win10 22H2 / macOS 14.3 | 我按Win11的方法教你，你发现用不了，浪费双方20分钟 |

## 3. 拍照与截图规范

### 手机拍屏幕的铁律：
1. **对焦** - 糊成毕加索油画的我看不懂
2. **光线** - 别背光，别反光，别拍成恐怖片
3. **全屏** - 我要看**整个屏幕**，不是只拍错误弹窗，上下文很重要
4. **别拍键盘** - 除非问题是键盘真的着火了

### 截图工具推荐（免费的，不用谢）：
- Win: Win+Shift+S 或 ShareX
- Mac: Cmd+Shift+4
- 手机：电源键+音量键，别他妈用另一个手机拍

---

## 4. 禁止行为（违者直接拉黑）

### 🚫 高频傻逼问题 TOP 5

1. **"在吗？"**
   - 有话直说，我不在，我永远不在，但我会看消息。直接发问题。

2. **"你能帮我看看吗？很快的"**
   - 没有"很快"的电脑问题。你感觉很快是因为你不懂。

3. **"我按你说的做了，还是不行"**
   - **详细描述**怎么做的，出现了什么**新**的错误。这句话等于"医生我吃药了还是难受"，医生怎么知道你怎么吃的？

4. **"你是不是觉得我很笨？"**
   - 不，我觉得你很忙，忙到连百度都懒得用。

5. **"淘宝修电脑的说是XX问题"**
   - 淘宝客服的目标是让你下单，不是解决问题。别拿他们的诊断来质疑我。

- **准备好管理员密码** - 别他妈临时打电话问你妈
- **坐在电脑前** - 别去吃饭让我对着桌面发呆
- **别同时找两个人** - 多个"医生"一起动刀，死得更快

## 6. 费用与尊重

**免费帮忙 ≠ 24小时客服**

- 晚上11点后发消息，最好准备好红包
- 解决了问题，说句"谢谢"是最低要求
- "请喝奶茶"是基本礼仪，不是可选项目

好的提问 = 具体环境 + 明确现象 + 已做尝试 + 截图/照片
